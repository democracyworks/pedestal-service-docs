# pedestal-service-docs

A Leiningen plugin to generate pedestal service API docs

## Usage

Put `[pedestal-service-docs "0.1.0-SNAPSHOT"]` into the `:plugins` vector of your project.clj.

Generate some docs:

    $ lein pedestal-service-docs
